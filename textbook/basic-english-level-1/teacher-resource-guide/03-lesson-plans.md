# SKC TEACHER'S RESOURCE GUIDE

## Section 3: Sample Lesson Plans

[Illustration: A teacher's planning notebook open on a desk, showing a simple timetable grid with lesson stages written in, a clock and coffee cup nearby, flat-vector illustration.]

Each unit in this book is designed to fill one 90-minute lesson (or two 45-minute sessions). Below is a general lesson plan template, followed by three fully worked example lesson plans for Units 1, 9, and 17.

---

## General Lesson Plan Template (90 Minutes)

| Time | Stage | Activity |
|---|---|---|
| 0–5 min | Warm-Up | Greet students; do the unit's Warm-Up activity |
| 5–20 min | Vocabulary Presentation | Present new words using pictures/realia; drill pronunciation |
| 20–30 min | Illustrated Words | Reinforce key words with the illustration poster |
| 30–40 min | Conversation | Model the dialogue; students practice in pairs |
| 40–50 min | Grammar | Present the grammar point; guided practice |
| 50–55 min | Pronunciation | Drill target sounds |
| 55–65 min | Speaking | Pair/group speaking activity |
| 65–70 min | Listening | Read the listening script; check answers |
| 70–75 min | Reading | Read the passage; answer comprehension questions |
| 75–85 min | Writing & Practice | Complete writing and practice exercises |
| 85–90 min | Game, Review & Homework | Play the unit game; review key points; assign homework |

*Note: The Mini Assessment can be given as a short quiz at the start of the next lesson, as a review of the previous unit.*

## Sample Lesson Plan 1 — Unit 1: Hello

**Objective:** Students will greet others appropriately for time of day and use basic polite expressions.

| Time | Stage | Teacher Action | Student Action |
|---|---|---|---|
| 0–5 | Warm-Up | Show the four-panel greeting picture; ask "What do you think they say?" | Guess greetings in Somali or English |
| 5–20 | Vocabulary | Present hello, hi, good morning/afternoon/evening/night using the sun-cycle illustration; drill chorally and individually | Repeat each word 3 times |
| 20–30 | Illustrated Words | Point to the times-of-day chart; students match greeting to time | Point and say the greeting |
| 30–40 | Conversation | Model "Meeting at School" dialogue with a volunteer; then pairs practice | Practice dialogue in pairs, swap roles |
| 40–50 | Grammar | Present the greeting-by-time-of-day table; do the practice exercise together | Complete grammar practice individually, then check as a class |
| 50–55 | Pronunciation | Drill the /h/ sound and "thank you" linking | Repeat after teacher |
| 55–65 | Speaking | Greeting Chain activity — students greet 3 classmates | Walk around, greet classmates |
| 65–70 | Listening | Read the listening script twice; students answer questions | Listen and write answers |
| 70–75 | Reading | Students read the Amina passage silently, then aloud; answer questions | Read and answer |
| 75–85 | Writing | Complete tracing, copying, and fill-in-the-blank exercises | Write in notebooks |
| 85–90 | Game & Review | Play Greeting Ball Toss; review key vocabulary | Participate; recall today's words |

**Homework:** Greet three people at home; write five sentences using unit vocabulary.

## Sample Lesson Plan 2 — Unit 9: Days and Months

**Objective:** Students will name the days of the week and months of the year and use "on" and "in" correctly.

| Time | Stage | Teacher Action | Student Action |
|---|---|---|---|
| 0–5 | Warm-Up | Show the weekly calendar strip; ask "What day is today?" | Answer using known words or Somali |
| 5–20 | Vocabulary | Present days and months using the week-cycle wheel; drill pronunciation, especially stress patterns | Repeat and clap on stressed syllables |
| 20–30 | Illustrated Words | Use the circular week-wheel poster to reinforce daily associations | Point to each day and describe a typical activity |
| 30–40 | Conversation | Model "Talking About the Week" dialogue; pairs practice with their own info | Practice in pairs |
| 40–50 | Grammar | Present "on" + day, "in" + month; complete practice sentences | Fill in blanks, check as a class |
| 50–55 | Pronunciation | Drill day/month stress patterns as a chant | Chant the days and months in order |
| 55–65 | Speaking | Students share their weekly schedule and favorite month | Speak in pairs, then share with class |
| 65–70 | Listening | Read the listening script twice; answer questions | Listen and write answers |
| 70–75 | Reading | Read the passage about favorite day/month; answer questions | Read and answer |
| 75–85 | Writing | Complete tracing, ordering days/months, and fill-in-blank exercises | Write in notebooks |
| 85–90 | Game & Review | Days in Order Race; review key vocabulary | Participate in teams |

**Homework:** Write the days and months in order; write one sentence about each day's activity this week.

## Sample Lesson Plan 3 — Unit 17: Transport

**Objective:** Students will name transport types and describe how they travel using "by."

| Time | Stage | Teacher Action | Student Action |
|---|---|---|---|
| 0–5 | Warm-Up | Show the street scene picture; ask "How many types of transport can you name?" | Name known transport words |
| 5–20 | Vocabulary | Present transport vocabulary using the ways-to-travel poster; drill pronunciation | Repeat each word |
| 20–30 | Illustrated Words | Compare transport methods by speed/distance using the poster | Discuss which transport is fastest/slowest |
| 30–40 | Conversation | Model "Taking the Bus" dialogue; pairs role-play with different destinations/fares | Practice in pairs |
| 40–50 | Grammar | Present "by" + transport structure; complete practice sentences | Fill in blanks, check as a class |
| 50–55 | Pronunciation | Drill the /b/ sound in "bus," "bicycle," "boat" | Repeat after teacher |
| 55–65 | Speaking | Students describe their daily journey to SKC | Speak in pairs, then share with class |
| 65–70 | Listening | Read the listening script twice; answer questions | Listen and write answers |
| 70–75 | Reading | Read the passage about traveling to SKC; answer questions | Read and answer |
| 75–85 | Writing | Complete tracing, fill-in-blank, and matching exercises | Write in notebooks |
| 85–90 | Game & Review | Traffic Light Game; review key vocabulary | Act out red/yellow/green |

**Homework:** Write how you travel to five different places; draw and label your favorite transport type.

### Adapting These Plans

Teachers can use Sample Lesson Plans 1–3 as models for planning **any** of the 20 units — simply substitute the vocabulary, conversation, grammar point, and activities from the relevant unit into the same 90-minute structure.
